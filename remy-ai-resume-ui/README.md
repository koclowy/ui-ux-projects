##  Overview

Remy addresses a major challenge faced by job seekers:  
**creating resumes that truly match their skills, personality, and job descriptions.**

The platform integrates:
- AI-powered resume generation  
- Personality profiling through guided questions  
- Job-description matching  
- Cover letter generation  
- Career insights dashboard  

According to the project documentation, Remy combines **profile building, personality assessment, and resume tailoring** within a unified AI-driven ecosystem.

## Design Process

### 1. **Research**
Based on market/industry analysis in the documentation:
- Studied HRTech competitors (Resume.io, Rezi, Kickresume)  
- Analysed pain points: generic resumes, resume fatigue, job mismatch, lack of personalization  
- Identified opportunity in **ethical, inclusive, and personalised AI career tools**  

### 2. **User Needs & Insight Mapping**
Key needs extracted from the business plan:
- Authentic self-expression  
- Clearer articulation of strengths  
- Job-role alignment  
- Faster resume customisation process  

### 3. **Wireframes (Figma)**
Core layouts include:
- Onboarding & profile setup  
- Personality assessment screens  
- Resume generation interface  
- Resume preview & export  
- Dashboard with job insights  

### 4. **High-Fidelity UI**
Design focuses on:
- Clean, modern UI with neutral tones  
- Friendly typography for young job-seekers  
- Inclusive design patterns  
- Professional yet warm visual identity  
- Modular card components for profile data  

### 5. **Interactive Prototype**
Prototype demonstrates:
- Chatbot personality discovery flow  
- Resume generation steps  
- Job-description matching  
- Dashboard navigation

## Learnings

From this project, I learned how to:

- Design AI-driven user flows for career tools
